# HitSpan

HitSpan is a lightweight, client-side HUD mod for that displays:

- **Hit range**, calculated using Minecraftâ€™s own ray logic
- **Knockback distance**, measured in real time after confirmed hits

The goal of the mod is to stay simple and accurate.

## Features
- Accurate hit range calculation
- Real knockback tracking over multiple ticks
- Separate HUD boxes for range and knockback
- Client-side only
- Safe for multiplayer and modpacks

## Current Status
This is **v1.1**, and it is intentionally minimal.

There is now **a config menu**!  
HUD customization (position, toggles) and General Customization (players only, hud decay/reset time.
More config's are planned in the future, along with a better config menu.

## Compatibility
- Minecraft **1.8.9**
- Forge **11.15.1.2318**
- Client-side only

## Installation
1. Install Minecraft Forge for 1.8.9
2. Drop the HitSpan `.jar` into your `mods` folder
3. Launch Minecraft
